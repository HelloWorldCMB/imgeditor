imgeditor
=========

A set of tools for editor binary images.

# Compile and run on host

```console
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```

And we can do some builtin tests.

```console
$ make test
Running tests...
...
100% tests passed, 0 tests failed out of 8

Total Test time (real) =  16.61 sec
```
